**Step 1: Write the Server Action**

Modify `app/actions/artifacts.ts` to add a new function:

```typescript
export async function revalidateArtifacts() {
  revalidatePath("/artifacts");
}
```

**Step 2: Trigger Cache Invalidation on Generation Success**

Modify `app/page.tsx`:
1. Import the action: `import { revalidateArtifacts } from "@/app/actions/artifacts";`
2. Update `handleSubmit` function to call the invalidation when the pipeline loop finishes successfully:

```typescript

try {
  for (let i = 0; i < total; i++) {
    // ...

  // Trigger cache invalidation since backend silently saved artifacts

  setIsDone(true);
}
```
