# Scaffold FastAPI Application (SOLO-101) - Design Document

## Goal Description

The goal of this task is to scaffold a new FastAPI application to serve as the backend API for the `note-taker` application. This API will be consumed by the Next.js frontend, so it must define a clear entry point and be configured properly with CORS.

## Proposed Architecture

We will implement **Approach B: API Sub-package** to integrate the FastAPI application directly into the existing `note_taker` Python package for better maintainability and scalability.

### 1. API Directory Structure

We will create a new directory `src/note_taker/api/` and populate it with the following core files:

*   **`src/note_taker/api/__init__.py`**: Marks the directory as a Python sub-package.
*   **`src/note_taker/api/main.py`**: The entry point for the FastAPI application. It will instantiate the `FastAPI` app, configure middleware (specifically CORS), and include API routers.
*   **`src/note_taker/api/routes.py`**: Contains the initial set of API routers (e.g., a simple health check `/health` endpoint to start).

### 2. CORS Configuration

To allow the Next.js frontend to communicate with this new backend, we will configure the `CORSMiddleware` in `main.py`.

*   **Allowed Origins**: Initially set to `http://localhost:3000` (development frontend). We should ideally allow this to be configurable via environment variables in the future.
*   **Allowed Methods**: `["*"]` (Allow all standard methods: GET, POST, PUT, DELETE, OPTIONS, etc.).
*   **Allowed Headers**: `["*"]` (Allow all headers).
*   **Allow Credentials**: `True` (typically required if we introduce authentication cookies later).

### 3. Dependencies

We will need to ensure `fastapi` and an ASGI server like `uvicorn` are added to the project's dependencies in `pyproject.toml` if they are not already present.

## User Review Required

Since this design was already discussed and approved in the brainstorming phase, no further user review is required on the core structure. The implementation plan will proceed based on this document.
